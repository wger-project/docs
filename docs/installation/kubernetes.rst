.. _kubernetes:

Kubernetes
==========

Helm charts for deploying wger to a Kubernetes cluster are available in the
`wger-project/helm-charts <https://github.com/wger-project/helm-charts>`_
repository.

The charts are maintained by a community contributor and kept current with
new wger releases. For installation and configuration details, see the repository's
`README <https://github.com/wger-project/helm-charts/blob/main/README.md>`_.

For issues with the charts themselves (templating, values, install errors),
open an issue against
`wger-project/helm-charts <https://github.com/wger-project/helm-charts/issues>`_.
For issues with wger itself, open them against the main `wger repository
<https://github.com/wger-project/wger/issues>`_ instead.
