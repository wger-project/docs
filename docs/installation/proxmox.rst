Proxmox
=======

There is a community-maintained installation script for Proxmox VE:

  https://community-scripts.org/scripts/wger

The script lives in the
`community-scripts/ProxmoxVE <https://github.com/community-scripts/ProxmoxVE>`_
repository (``ct/wger.sh`` and ``install/wger-install.sh``).

This script is **not maintained by the wger project**. We can't guarantee that
it stays up to date with current wger releases, and we can't provide support
for issues caused by the script itself. For problems with the script, open an
issue against
`community-scripts/ProxmoxVE <https://github.com/community-scripts/ProxmoxVE/issues>`_.

For a setup that we test and support, see :doc:`docker`.
